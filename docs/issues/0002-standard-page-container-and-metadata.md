# 标准业务页容器与页面元数据

**类型：** AFK  
**标签：** `ready-for-agent`

## What to build

将工作台页面注册表扩展为菜单、路由、工作台标签、标题和面包屑的唯一来源，并提供轻量 `PageContainer`。该容器自动呈现标准业务页的标题与面包屑，业务操作通过运行时 `extra` 注入。将系统字典占位页作为首个标准业务页接入，验证这一完整路径；不实现系统字典的实际 CRUD。

## Acceptance criteria

- [ ] 页面元数据可一致驱动菜单、路由、工作台标签、标题、面包屑和页面类型。
- [ ] 标准业务页中的 `PageContainer` 自动展示标题和面包屑，并可展示业务页面提供的 `extra` 内容。
- [ ] 系统字典占位页以标准业务页方式呈现，但不新增真实业务数据、接口或 CRUD。
- [ ] 现有工作台标签行为保持：首页不可关闭，已访问的非首页页面可关闭。
- [ ] 为元数据和 `PageContainer` 增加外部行为测试，并通过类型检查、单元测试和生产构建。

## Blocked by

None - can start immediately.
