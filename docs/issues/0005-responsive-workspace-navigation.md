# 工作台移动端抽屉导航

**类型：** AFK  
**标签：** `ready-for-agent`

## What to build

改造工作台导航的响应式行为。桌面端继续使用固定、可折叠的侧栏和工作台标签；窄屏端使用抽屉承载菜单，通过顶栏入口打开，并在用户选择菜单项后自动关闭。保持现有导航路径、工作台标签与登录会话行为。

## Acceptance criteria

- [ ] 桌面端保持当前固定侧栏、折叠能力和工作台标签体验。
- [ ] 窄屏端可从顶栏打开抽屉导航，菜单不再静态占满页面。
- [ ] 用户在窄屏端选择菜单项后，路由跳转且抽屉自动关闭。
- [ ] 工作台标签的打开、切换和关闭行为不因响应式改造而回退。
- [ ] 交互测试覆盖抽屉打开、菜单选择后的关闭及桌面标签行为，并通过类型检查、单元测试和生产构建。

## Blocked by

None - can start immediately.
