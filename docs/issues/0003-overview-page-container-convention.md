# 概览页容器规范

**类型：** AFK  
**标签：** `ready-for-agent`

> 已被 ADR-0006 取代：概览页与其他业务页均按自身内容组合使用 `BasicContainer`，不再存在 `PageContainer` 页头规范。

## What to build

在已有页面元数据和 `PageContainer` 基础上，支持概览页与标准业务页的不同呈现。将首页迁移到概览页规范：保持统一内容边距和容器能力，但不渲染重复的标题或面包屑页头，保留首页现有概览内容。

## Acceptance criteria

- [ ] 页面元数据可声明概览页类型，并由页面容器使用该类型决定页头呈现。
- [ ] 首页保留现有概览内容与统一内容边距，不显示冗余页头和面包屑。
- [ ] 标准业务页的页头行为不因概览页支持而改变。
- [ ] 路由级渲染测试覆盖概览页隐藏页头和标准业务页显示页头的可见行为。
- [ ] 类型检查、单元测试和生产构建通过。

## Blocked by

- 本地议题 0002：标准业务页容器与页面元数据。
