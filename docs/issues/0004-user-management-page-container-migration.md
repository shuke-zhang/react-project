# 用户管理页接入标准页面容器

**类型：** AFK  
**标签：** `ready-for-agent`

## What to build

将用户管理迁移到标准业务页容器。页面级标题和面包屑由 `PageContainer` 提供，“新增用户”通过 `extra` 运行时注入；全局顶栏只保留应用级操作。保留用户管理的查询、新增、编辑和删除体验及其既有数据访问契约。

## Acceptance criteria

- [ ] 用户管理页展示由页面元数据提供的标题和面包屑。
- [ ] “新增用户”位于用户管理页面的操作区，并仍可打开新增表单。
- [ ] 顶栏不再重复展示当前业务页面标题，仍保留全局控制和用户菜单。
- [ ] 用户查询、新增、编辑、删除及成功提示行为保持不变。
- [ ] 页面级交互测试覆盖迁移后的用户管理关键操作，并通过类型检查、单元测试和生产构建。

## Blocked by

- 本地议题 0002：标准业务页容器与页面元数据。
