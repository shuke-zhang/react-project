# Triage Labels

Skills 使用五个标准 triage 角色。本文档把这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker 中的 label | 含义 |
| ---------------------------- | ------------------------- | ---- |
| `needs-triage`               | `needs-triage`            | 需要维护者评估这个 issue |
| `needs-info`                 | `needs-info`              | 等待报告者补充更多信息 |
| `ready-for-agent`            | `ready-for-agent`         | 描述完整，适合 AFK agent 接手 |
| `ready-for-human`            | `ready-for-human`         | 需要人类实现 |
| `wontfix`                    | `wontfix`                 | 不会处理 |

当 skill 提到某个角色时，例如 "apply the AFK-ready triage label"，使用上表中对应的 label 字符串。
