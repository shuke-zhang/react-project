# 企业级 React 项目模板

本上下文记录企业级 React 项目模板的领域语言，用于保持文档、页面和测试描述一致。

## Language

**登录会话**:
用户通过 Blade 登录接口获得的访问状态，由访问令牌、刷新令牌和租户 ID 共同表示。
_Avoid_: 登录状态、会话信息

**访问令牌**:
用于访问受保护接口的短期凭证。
_Avoid_: token、业务 token

**刷新令牌**:
用于在访问令牌失效后换取新访问令牌的长期凭证。
_Avoid_: refresh token

**用户**:
后台系统中可登录或被管理的账号主体。
_Avoid_: 账号、成员

**用户状态**:
用户是否可正常使用系统的业务状态，当前包括启用和停用。
_Avoid_: 状态值、账号状态

**角色**:
用于表达用户职责或权限集合的业务标签。
_Avoid_: 权限组

**部门**:
用户所属的组织单元。
_Avoid_: 科室、组织

**系统字典**:
用于维护后台系统中可复用枚举、选项和值域说明的配置集合。
_Avoid_: 字典、数据字典、枚举表
