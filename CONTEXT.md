# 企业级 React 项目模板

本上下文记录企业级 React 项目模板的领域语言，用于保持文档、页面和测试描述一致。

## Language

**登录会话**:
用户通过 Blade 登录接口获得的访问状态，由访问令牌、刷新令牌和租户 ID 共同表示。
_Avoid_: 登录状态、会话信息

**访问令牌**:
用于访问受保护接口的短期凭证。
_Avoid_: token、业务 token

**刷新令牌**:
用于在访问令牌失效后换取新访问令牌的长期凭证。
_Avoid_: refresh token

**用户**:
后台系统中可登录或被管理的账号主体。
_Avoid_: 账号、成员

**用户状态**:
用户是否可正常使用系统的业务状态，当前包括启用和停用。
_Avoid_: 状态值、账号状态

**角色**:
用于表达用户职责或权限集合的业务标签。
_Avoid_: 权限组

**部门**:
用户所属的组织单元。
_Avoid_: 科室、组织

**系统字典**:
用于维护后台系统中可复用枚举、选项和值域说明的配置集合。
_Avoid_: 字典、数据字典、枚举表

**用户管理**:
后台系统中围绕用户查询、筛选、新增、编辑和删除的业务能力。
_Avoid_: 用户列表页面、用户 CRUD

**工作台标签**:
后台工作台中表示一次已访问页面，并支持在页面间切换和关闭的导航单元。
_Avoid_: 页签、选项卡

**基础内容容器**:
工作台业务页面用于获得统一白色内容背景、尺寸约束与滚动区域的通用容器。
_Avoid_: PageContainer、页面结构容器

**滚动区域**:
在基础内容容器中承载业务内容、提供跨浏览器统一滚动条外观的可滚动区域。
_Avoid_: 第三方滚动条、页面容器

**React 知识点页面**:
围绕一个明确 React API、Hook、组件模式或工程实践建立的独立路由页面，例如 `useState`、组件通信或 `useReducer`；同一页面从基础用法讲到进阶与企业实践。
_Avoid_: 综合案例页、基础与进阶拆页、视频配套页

**React 学习模块**:
工作台全局导航中的扁平 React 知识点菜单，每个菜单项通过独立子路由进入一个 React 知识点页面，不增加分类子菜单或内容区第二套导航。
_Avoid_: 学习中心、分类子菜单、视频目录、双侧栏布局

**在线练习区**:
React 学习案例中用于安全编辑、编译并预览 TSX 源码的浏览器沙箱，支持错误反馈和恢复默认代码。
_Avoid_: 在线编辑器、代码输入框、eval 运行区

**企业级扩展**:
React 知识点页面中的进阶教学层，用于说明该知识点在状态边界、组件契约、性能、错误处理或可测试性方面的真实项目约束。
_Avoid_: 独立高阶菜单、高级炫技、与当前知识点无关的大案例

**React 企业工程**:
与 React 核心知识明确分区的项目实践集合，仅纳入当前项目已经使用且高频的路由、异步请求、缓存、数据校验和测试能力。
_Avoid_: React 核心 API、第三方库百科、依赖堆砌

**React API 范围**:
React 学习模块覆盖 React 19 在当前 Vite 客户端项目中可实际使用的稳定 API，包括低频但仍有效的客户端专用 API；服务端 API、React Compiler、废弃 API和旧式 Class Component API不进入主课程。
_Avoid_: React 历史 API 全集、服务端 API、编译器课程、废弃 API 主课程
